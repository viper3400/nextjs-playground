# Project Setup

## Environment Variables
Find the `.env` configuration file in the root folder of the project.
It's recommended to copy this file into `.env.local`
(see [environment variables in Next.js](https://nextjs.org/docs/pages/building-your-application/configuring/environment-variables)
for further informations).

As this file may contain sensible data please make sure not to commit it into your repo.

### Create a NEXTAUTH_SECRET

You can run `openssl rand -base64 32` to quickly create a good value on the command line.

See [NextAuth.js options](https://next-auth.js.org/configuration/options) for more informations.
